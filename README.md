## Browser Extension

Save anything from the web directly to Memora.

### Download
[Click here to download](https://github.com/Vishalnaidu77/memora/releases/download/v1.0.0/memora-extension.zip)

### Installation Steps
1. Download and extract the ZIP file
2. Open Chrome → go to `chrome://extensions`
3. Enable **Developer Mode** (top right toggle)
4. Click **"Load unpacked"**
5. Select the extracted `memora-extension` folder
6. Memora icon will appear in your toolbar
7. Login to Memora first, then click the icon to save any page
